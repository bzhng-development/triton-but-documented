‘tt’ Dialect¶

The Triton IR in MLIR

Triton Dialect.

Dependent Dialects:

  • Arith:

    • addf, addi, andi, cmpf, cmpi, divf, fptosi, …

  • Math:

    • exp, sin, cos, log, …

  • StructuredControlFlow:

    • for, if, while, yield, condition

  • ControlFlow:

    • br, cond_br

TensorDescType¶

Tensor descriptor type (::mlir::triton::TensorDescType) in Triton IR type system

Syntax:

!tt.tensordesc<
  RankedTensorType   # blockType
>

A portable abstraction for TMA descriptors. This is the base tensor descriptor type for tiled tensor memory access.

For specialized access patterns like im2col, see TensorDescIm2ColType in the TritonNvidiaGPU dialect.
