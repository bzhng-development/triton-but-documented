class Mark¶

Decorator class for running and visualizing Triton kernel benchmarks.

Wraps a benchmarked function and one or more Benchmark configurations to automatically execute performance tests, generate plots, and export results to CSV. Typically instantiated via the perf_report() helper.

Parameters¶

fncallable

The Triton kernel or benchmark function to execute. It must accept arguments defined in the benchmark configurations (x_names, line_arg, and args).

benchmarksBenchmark or list of Benchmark

A single Benchmark instance or a list of instances defining the parameter sweeps, axis labels, and plot metadata.

Attributes¶

fncallable

The stored function to benchmark.

benchmarkslist of Benchmark

The stored list of benchmark configurations.

Methods¶

run(show_plots=False, print_data=False, save_path=’’, return_df=False, **kwargs)

Execute the benchmarks defined in the constructor.

show_plotsbool, optional

If True, display matplotlib windows during execution. Default is False.

print_databool, optional

If True, print the resulting DataFrame to stdout. Default is False.

save_pathstr, optional

Directory path to save generated plots (.png) and data (.csv). If provided, an results.html file summarizing the plots is also created. Default is empty string (no saving).

return_dfbool, optional

If True, return the pandas DataFrame(s) containing the results. Default is False.

**kwargsdict

Additional keyword arguments passed directly to the benchmark function fn.

pandas.DataFrame or list of pandas.DataFrame or None

If return_df is True, returns the result DataFrame for a single benchmark or a list of DataFrames for multiple benchmarks. Otherwise, returns None.

Notes¶

The benchmark function fn is expected to return a scalar (mean time) or a tuple of three values (mean, min, max). If a scalar is returned, min/max error bars will not be plotted.

The x-axis values are derived from Benchmark.x_vals. If multiple x_names are specified, x_vals elements can be scalars (broadcasted) or sequences matching the length of x_names.

Output files saved to save_path include: - {plot_name}.png: Line plot with optional error bands. - {plot_name}.csv: Tabular data with mean values. - results.html: Simple HTML page embedding all generated plots.

Examples¶

import triton
import triton.language as tl
from triton.testing import Benchmark, perf_report

@triton.jit
def vector_add_kernel(x_ptr, y_ptr, out_ptr, n,
                      BLOCK_SIZE: tl.constexpr):
    pid = tl.program_id(0)
    offs = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)
    mask = offs < n
    x = tl.load(x_ptr + offs, mask=mask)
    y = tl.load(y_ptr + offs, mask=mask)
    tl.store(out_ptr + offs, x + y, mask=mask)
