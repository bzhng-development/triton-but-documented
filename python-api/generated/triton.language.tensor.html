class triton.language.tensor(handle, type: dtype)¶

Represents an N-dimensional array of values or pointers.

tensor is the fundamental data structure in Triton programs. Most functions in triton.language operate on and return tensors.

Most of the named member functions here are duplicates of the free functions in triton.language. For example, triton.language.sqrt(x) is equivalent to x.sqrt().

tensor also defines most of the magic/dunder methods, so you can write x+y, x << 2, etc.

Constructors

__init__(handle, type: dtype)¶

Not called by user code. Tensors are typically created via helper functions such as triton.language.full(), triton.language.arange(), or by loading data from memory using triton.language.load().

Attributes

dtype¶

The scalar data type of the tensor elements (e.g., tl.float32).

shape¶

A tuple of constexpr integers representing the dimensions of the tensor.

numel¶

The total number of elements in the tensor.

type¶

The full tensor type, which may include block shape information.

T¶

Property that returns a transposed view of a 2D tensor. Transposition is handled by the AST visitor; direct access raises an error.

Operations

The tensor class supports standard Python arithmetic, bitwise, and comparison operators, which are lowered to GPU instructions:

  • Arithmetic: +, -, *, /, //, %

  • Bitwise: &, |, ^, <<, >>, ~

  • Comparison: ==, !=, <, <=, >, >=

  • Unary: -, not

Member Functions

The following methods are available as members of tensor. They are equivalent to the free functions in triton.language with the tensor passed as the first argument.

broadcast_to(*shape)¶

Broadcasts the tensor to the specified shape.

trans(*dims)¶

Permutes the dimensions. If no arguments are provided, swaps the last two dimensions (2D transpose).

permute(*dims)¶

Permutes the dimensions according to the provided order.

split()¶

Splits the tensor in two along the last dimension (which must have size 2).

view(*shape)¶

Returns a tensor with the same elements but a different shape. Deprecated; use reshape() instead.

reshape(*shape, can_reorder=False)¶

Returns a tensor with the same number of elements but a new shape.

expand_dims(axis)¶

Inserts a new length-1 dimension at the specified axis.

cast(dtype, fp_downcast_rounding=None, bitcast=False)¶

Casts the tensor to the specified data type.

to(dtype, fp_downcast_rounding=None, bitcast=False)¶

Alias for cast().

store(pointer, value, mask=None, boundary_check=(), cache_modifier='', eviction_policy='')¶

Stores the tensor to memory at the given pointer.

advance(offsets)¶

Advances a block pointer by the given offsets.

atomic_cas(cmp, val, sem=None, scope=None)¶

Performs an atomic compare-and-swap.

atomic_xchg(val, mask=None, sem=None, scope=None)¶

Performs an atomic exchange.

atomic_add(val, mask=None, sem=None, scope=None)¶

Performs an atomic add.

atomic_max(val, mask=None, sem=None, scope=None)¶

Performs an atomic max.

atomic_min(val, mask=None, sem=None, scope=None)¶

Performs an atomic min.

atomic_and(val, mask=None, sem=None, scope=None)¶

Performs an atomic bitwise AND.

atomic_or(val, mask=None, sem=None, scope=None)¶

Performs an atomic bitwise OR.

atomic_xor(val, mask=None, sem=None, scope=None)¶

Performs an atomic bitwise XOR.

exp()¶

Computes the exponential of each element.

log()¶

Computes the natural logarithm of each element.

cos()¶

Computes the cosine of each element.

sin()¶

Computes the sine of each element.

sqrt()¶

Computes the square root of each element.

rsqrt()¶

Computes the reciprocal square root of each element.

abs()¶

Computes the absolute value of each element.

reduce(axis, combine_fn, keep_dims=False)¶

Reduces the tensor along an axis using a custom combine function.

associative_scan(axis, combine_fn, reverse=False)¶

Performs an associative scan (prefix sum/product) along an axis.

gather(indices, axis)¶

Gathers slices from the tensor along an axis.

histogram(num_bins)¶

Computes a histogram of the tensor values.

cdiv(div)¶

Computes ceiling division.

sigmoid()¶

Computes the sigmoid function.

softmax(dim=None, keep_dims=False, ieee_rounding=False)¶

Computes the softmax along a dimension.

ravel()¶

Flattens the tensor to 1D.

max(axis=None, return_indices=False, return_indices_tie_break_left=True, keep_dims=False)¶

Computes the maximum value along an axis.

argmax(axis, tie_break_left=True, keep_dims=False)¶

Returns the indices of the maximum value.

min(axis=None, return_indices=False, return_indices_tie_break_left=True, keep_dims=False)¶

Computes the minimum value along an axis.

argmin(axis, tie_break_left=True, keep_dims=False)¶

Returns the indices of the minimum value.

sum(axis=None, keep_dims=False, dtype=None)¶

Computes the sum of elements along an axis.

xor_sum(axis=None, keep_dims=False)¶

Computes the bitwise XOR sum along an axis.

reduce_or(axis=None, keep_dims=False)¶

Computes the bitwise OR reduction along an axis.

cumsum(axis=0, reverse=False)¶

Computes the cumulative sum along an axis.

cumprod(axis=0, reverse=False)¶

Computes the cumulative product along an axis.

sort(dim=None, descending=False)¶

Sorts the tensor along a dimension.

flip(dim=None)¶

Reverses the order of elements along a dimension.

Examples

import triton
import triton.language as tl

@triton.jit
def kernel(x_ptr, y_ptr, BLOCK_SIZE: tl.constexpr):
    # Load a block of data into a tensor
    x = tl.load(x_ptr + tl.arange(0, BLOCK_SIZE))

    # Perform element-wise operations using tensor methods
    y = x.sqrt() * 2.0

    # Reduce the tensor to a single value
    sum_x = tl.sum(x, axis=0)

    # Store the result
    tl.store(y_ptr + tl.arange(0, BLOCK_SIZE), y)
