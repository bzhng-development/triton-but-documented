philox¶

triton.language.philox(seed, c0, c1, c2, c3, n_rounds=10)¶

Apply the Philox pseudo-random number generator to a state vector.

This function performs n_rounds of the Philox encryption algorithm on the input state (c0, c1, c2, c3) using the provided seed as the key. It supports both 32-bit and 64-bit integer arithmetic, automatically casting inputs to the appropriate precision based on the dtype of c0.

Parameters¶

seedtl.tensor

The seed value used to initialize the generator key. Must be an integer type. If 64-bit, it is split into high and low 32-bit parts for 32-bit modes, or used directly as the low part for 64-bit modes.

c0tl.tensor

First element of the state vector. Determines the output precision (uint32 or uint64).

c1tl.tensor

Second element of the state vector.

c2tl.tensor

Third element of the state vector.

c3tl.tensor

Fourth element of the state vector.

n_roundstl.constexpr, optional

Number of encryption rounds to perform. Default is 10.

Returns¶

tl.tensor

A tuple of four tensors (c0, c1, c2, c3) representing the transformed state vector after applying the Philox rounds. The dtype matches the input dtype of c0.

Notes¶

This is a low-level primitive typically used by higher-level random number generation functions like randint4x or rand4x. Direct usage requires managing the state vector and offsets manually.

The function asserts that the bitwidth of c0 is either 32 or 64. Mixed precision or unsupported bitwidths will raise a compilation error.

Examples¶

Generate a block of random uint32 values using explicit state management:

import triton
import triton.language as tl

@triton.jit
def my_kernel(seed_ptr, output_ptr, BLOCK_SIZE: tl.constexpr):
    pid = tl.program_id(0)
    offset = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)

    # Initialize state with offsets
    c0 = offset
    c1 = tl.zeros([BLOCK_SIZE], dtype=tl.uint32)
    c2 = tl.zeros([BLOCK_SIZE], dtype=tl.uint32)
    c3 = tl.zeros([BLOCK_SIZE], dtype=tl.uint32)

    seed = tl.load(seed_ptr)

    # Apply Philox rounds
    r0, r1, r2, r3 = tl.philox(seed, c0, c1, c2, c3)

    # Use the first stream of random numbers
    tl.store(output_ptr + offset, r0)
