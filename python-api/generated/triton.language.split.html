split¶

triton.language.split(a, _semantic=None, _generator=None)¶

Split a tensor in two along its last dimension, which must have size 2.

Parameters¶

atensor

The tensor to split. The last dimension must have size 2.

Returns¶

tuple[tensor, tensor]

A tuple of two tensors. Each output tensor has the same shape as the input except the last dimension is removed. If the input is 1D with shape (2,), returns two scalars.

Notes¶

split is the inverse of join(). Given two tensors of shape (4, 8), join() produces a tensor of shape (4, 8, 2), and split() recovers the original two tensors.

Triton requires tensors to have power-of-two sizes. To split into more than two pieces, use multiple calls to this function (possibly combined with reshape()).

This function can also be called as a member function on tensor, as x.split() instead of split(x).

Examples¶

Split a 3D tensor along its last dimension:

@triton.jit
def kernel(...):
    x = tl.full((4, 8, 2), 1.0, dtype=tl.float32)
    left, right = tl.split(x)
    # left.shape == (4, 8), right.shape == (4, 8)

Split a 1D tensor (returns scalars):

@triton.jit
def kernel(...):
    x = tl.arange(0, 2).to(tl.float32)
    a, b = tl.split(x)
    # a and b are scalars

Using as a member function:

@triton.jit
def kernel(...):
    x = tl.full((4, 8, 2), 1.0, dtype=tl.float32)
    left, right = x.split()
