triton.language.rsqrt
=====================

.. currentmodule:: triton.language

.. function:: rsqrt(x, _semantic=None)

   Computes the element-wise inverse square root of :code:`x`.

   Parameters
   ----------
   x : Block
       The input tensor. Must be a floating-point type.

   Returns
   -------
   Block
       A tensor containing :code:`1 / sqrt(x)` for each element in :code:`x`.
       The output has the same shape and dtype as the input.

   Notes
   -----
   This function is only supported for floating-point types (:code:`fp32`, :code:`fp64`).

   This function can also be called as a member function on :py:class:`tensor`,
   as :code:`x.rsqrt()` instead of :code:`tl.rsqrt(x)`.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def kernel(x_ptr, y_ptr, BLOCK_SIZE: tl.constexpr):
          x = tl.load(x_ptr + tl.arange(0, BLOCK_SIZE))
          y = tl.rsqrt(x)
          tl.store(y_ptr + tl.arange(0, BLOCK_SIZE), y)
