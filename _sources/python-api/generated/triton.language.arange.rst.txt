triton.language.arange
======================

.. function:: triton.language.arange(start, end, _semantic=None)

   Returns contiguous values within the half-open interval :code:`[start, end)`.

   Parameters
   ----------
   start : int32
       Start of the interval. Must be a power of two.
   end : int32
       End of the interval. Must be a power of two greater than :code:`start`.

   Returns
   -------
   tensor
       1-D tensor of contiguous values from :code:`start` to :code:`end - 1`.

   Notes
   -----
   :code:`end - start` must be less than or equal to :code:`TRITON_MAX_TENSOR_NUMEL = 1048576`.

   Both :code:`start` and :code:`end` must be powers of two, which is required for Triton's block memory model.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def kernel(
          output_ptr,
          BLOCK_SIZE: tl.constexpr,
      ):
          # Create indices from 0 to BLOCK_SIZE - 1
          offsets = tl.arange(0, BLOCK_SIZE)
          # Use offsets to compute memory addresses
          tl.store(output_ptr + offsets, offsets)

      # Launch kernel with BLOCK_SIZE = 128 (must be power of two)
      kernel[(1,)](output, BLOCK_SIZE=128)
