exp2
====

.. currentmodule:: triton.language

.. function:: exp2(x, _semantic=None)

   Computes the element-wise exponential (base 2) of :code:`x`.

   Parameters
   ----------
   x : Block
       The input tensor. Must have floating-point dtype (:code:`fp32` or :code:`fp64`).

   Returns
   -------
   Block
       A tensor of the same shape as :code:`x` containing :code:`2^x` for each element.

   Notes
   -----
   This function is only supported for floating-point types (:code:`fp32`, :code:`fp64`).

   This function can also be called as a member function on :py:class:`tensor`,
   as :code:`x.exp2()` instead of :code:`exp2(x)`.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def kernel(X, Y, BLOCK_SIZE: tl.constexpr):
          offsets = tl.arange(0, BLOCK_SIZE)
          x = tl.load(X + offsets)
          y = tl.exp2(x)  # Compute 2^x element-wise
          tl.store(Y + offsets, y)
