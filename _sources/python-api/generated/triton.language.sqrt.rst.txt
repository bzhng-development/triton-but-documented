triton.language.sqrt
====================

.. currentmodule:: triton.language

.. function:: sqrt(x, _semantic=None)

   Computes the element-wise square root of :code:`x`.

   Parameters
   ----------
   x : tl.tensor
       The input tensor. Must have floating-point dtype (:code:`fp32` or :code:`fp64`).

   Returns
   -------
   tl.tensor
       A tensor of the same shape as :code:`x` containing the square root of each element.

   Notes
   -----
   This function can also be called as a member function on :py:class:`tensor`,
   as :code:`x.sqrt()` instead of :code:`tl.sqrt(x)`.

   Only floating-point types (:code:`fp32`, :code:`fp64`) are supported.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def sqrt_kernel(x_ptr, y_ptr, n, BLOCK_SIZE: tl.constexpr):
          pid = tl.program_id(0)
          offsets = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)
          mask = offsets < n
          x = tl.load(x_ptr + offsets, mask=mask)
          y = tl.sqrt(x)
          tl.store(y_ptr + offsets, y, mask=mask)
