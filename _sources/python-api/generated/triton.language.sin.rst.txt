.. py:function:: sin(x, _semantic=None)

   Computes the element-wise sine of :code:`x`.

   Parameters
   ----------
   x : tl.tensor
       The input tensor. Must be of floating-point type (:code:`fp32` or :code:`fp64`).

   Returns
   -------
   tl.tensor
       A tensor containing the sine of each element in :code:`x`.

   Notes
   -----
   This function can also be called as a member function on :py:class:`tensor`,
   as :code:`x.sin()` instead of :code:`tl.sin(x)`.

   The operation is only supported for floating-point dtypes. Passing integer
   types will result in a compilation error.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def kernel(x_ptr, y_ptr, BLOCK_SIZE: tl.constexpr):
          pid = tl.program_id(axis=0)
          offs = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)
          x = tl.load(x_ptr + offs)
          y = tl.sin(x)
          tl.store(y_ptr + offs, y)
