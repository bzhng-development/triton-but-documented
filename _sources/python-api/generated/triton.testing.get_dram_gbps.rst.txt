get_dram_gbps
=============

.. currentmodule:: triton.testing

.. autofunction:: get_dram_gbps

Return the theoretical DRAM bandwidth of the active GPU device in gigabytes per second (GB/s).

The bandwidth is calculated using the device's memory clock rate and memory bus width, assuming a double data rate (DDR) interface.

Parameters
----------
device : int, optional
    The CUDA device ID to query. If ``None``, the currently active device is used.

Returns
-------
bw_gbps : float
    The peak DRAM bandwidth in GB/s.

Notes
-----
The calculation follows the formula:

.. math::

   \text{Bandwidth (GB/s)} = \frac{\text{Memory Clock (kHz)} \times \text{Bus Width} \times 2}{10^6 \times 8}

where the factor of 2 accounts for double data rate (DDR) signaling, and division by 8 converts bits to bytes.

Examples
--------
.. code-block:: python

    import triton
    import triton.testing

    # Get DRAM bandwidth for the current active device
    bandwidth = triton.testing.get_dram_gbps()
    print(f"Peak DRAM Bandwidth: {bandwidth:.2f} GB/s")

    # Get DRAM bandwidth for a specific device (e.g., device 1)
    bandwidth_dev1 = triton.testing.get_dram_gbps(device=1)
    print(f"Device 1 Bandwidth: {bandwidth_dev1:.2f} GB/s")
