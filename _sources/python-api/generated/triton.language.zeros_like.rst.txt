zeros_like
============

.. currentmodule:: triton.language

.. function:: zeros_like(input)

   Returns a tensor of zeros with the same shape and data type as the input tensor.

   Parameters
   ----------
   input : tl.tensor
       Input tensor whose shape and dtype are used to create the output.

   Returns
   -------
   tl.tensor
       Tensor filled with zeros, having the same shape and dtype as ``input``.

   Notes
   -----
   This function is equivalent to ``tl.zeros(input.shape, input.dtype)``.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def kernel(x_ptr, out_ptr, BLOCK_SIZE: tl.constexpr):
          pid = tl.program_id(0)
          offs = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)
          x = tl.load(x_ptr + offs)
          zeros = tl.zeros_like(x)
          tl.store(out_ptr + offs, zeros)
