floor
=====

Compute the element-wise floor of the input tensor.

For each element :code:`x[i]`, returns the largest integer less than or equal to :code:`x[i]`.

Parameters
----------
x : tl.tensor
    Input tensor. Must have floating point dtype (:code:`fp32` or :code:`fp64`).

Returns
-------
tl.tensor
    Tensor of the same shape as :code:`x` containing the floor values.

Notes
-----
This function can also be called as a member function on :py:class:`tensor`,
as :code:`x.floor()` instead of :code:`tl.floor(x)`.

Only floating point types are supported (:code:`fp32`, :code:`fp64`).

Examples
--------
.. code-block:: python

    @triton.jit
    def kernel(x_ptr, y_ptr, BLOCK_SIZE: tl.constexpr):
        x = tl.load(x_ptr + tl.arange(0, BLOCK_SIZE))
        y = tl.floor(x)
        tl.store(y_ptr + tl.arange(0, BLOCK_SIZE), y)
