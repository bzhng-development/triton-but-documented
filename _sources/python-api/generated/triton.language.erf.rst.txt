erf
===

.. currentmodule:: triton.language

.. function:: erf(x, _semantic=None)

   Computes the element-wise error function of :code:`x`.

   The error function is defined as:

   .. math::

      \text{erf}(x) = \frac{2}{\sqrt{\pi}} \int_0^x e^{-t^2} dt

   Parameters
   ----------
   x : Block
      The input tensor. Must be of floating-point type (:code:`fp32` or :code:`fp64`).

   Returns
   -------
   Block
      A tensor of the same shape as :code:`x` containing the error function of each element.

   Notes
   -----
   This function can also be called as a member function on :py:class:`tensor`,
   as :code:`x.erf()` instead of :code:`erf(x)`.

   Supported dtypes: :code:`fp32`, :code:`fp64`.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def erf_kernel(x_ptr, y_ptr, BLOCK_SIZE: tl.constexpr):
          pid = tl.program_id(0)
          offsets = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)
          x = tl.load(x_ptr + offsets)
          y = tl.erf(x)
          tl.store(y_ptr + offsets, y)
