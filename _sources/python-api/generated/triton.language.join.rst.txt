.. py:function:: triton.language.join(a, b, _semantic=None)

   Join the given tensors in a new, minor dimension.

   For example, given two tensors of shape ``(4, 8)``, produces a new tensor of
   shape ``(4, 8, 2)``. Given two scalars, returns a tensor of shape ``(2,)``.

   The two inputs are broadcasted to be the same shape.

   To join more than two elements, use multiple calls to this function. This
   reflects the constraint in Triton that tensors must have power-of-two sizes.

   :py:func:`split` is the inverse of this function.

   Parameters
   ----------
   a : tensor
       The first input tensor.
   b : tensor
       The second input tensor.

   Returns
   -------
   tensor
       A tensor with the same shape as the broadcasted inputs, plus a new
       minor dimension of size 2.

   Examples
   --------
   .. code-block:: python

      @triton.jit
      def kernel(ptr_a, ptr_b, ptr_out, BLOCK_SIZE: tl.constexpr):
          a = tl.load(ptr_a + tl.arange(0, BLOCK_SIZE))
          b = tl.load(ptr_b + tl.arange(0, BLOCK_SIZE))
          # Join two 1D tensors of shape [BLOCK_SIZE] into a 2D tensor [BLOCK_SIZE, 2]
          c = tl.join(a, b)
          tl.store(ptr_out + tl.arange(0, BLOCK_SIZE * 2), c)
