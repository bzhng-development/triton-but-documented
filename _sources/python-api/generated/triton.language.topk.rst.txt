triton.language.topk
====================

.. currentmodule:: triton.language

.. function:: topk(x, k, dim=None, descending=True)

    Returns the :code:`k` largest (or smallest) elements of the input tensor along the specified dimension.

    The elements are returned in sorted order (largest first when :code:`descending=True`).

    Parameters
    ----------
    x : tensor
        The input tensor.
    k : constexpr
        The number of top elements to return. Must be a power of two.
    dim : constexpr, optional
        The dimension along which to find the top :code:`k` elements. If None, uses the last dimension. Currently only the last dimension is supported.
    descending : constexpr, optional
        If True (default), returns :code:`k` largest elements. If False, returns :code:`k` smallest elements.

    Returns
    -------
    tensor
        A tensor containing the :code:`k` largest (or smallest) elements along the specified dimension, in sorted order.

    Notes
    -----
    The value of :code:`k` must be a power of two for the underlying bitonic sort algorithm to work correctly.

    Only the last (minor) dimension is currently supported for the :code:`dim` parameter.

    Examples
    --------
    Get top 4 elements from a 1D tensor:

    .. code-block:: python

        import triton.language as tl

        x = tl.arange(0, 16)
        top4 = tl.topk(x, 4)  # Returns [15, 14, 13, 12]

    Get bottom 4 elements (smallest):

    .. code-block:: python

        x = tl.arange(0, 16)
        bottom4 = tl.topk(x, 4, descending=False)  # Returns [0, 1, 2, 3]
