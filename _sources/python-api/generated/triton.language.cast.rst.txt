.. py:function:: triton.language.cast(input, dtype, fp_downcast_rounding=None, bitcast=False)

   Casts a tensor to the given ``dtype``.

   :param input: The input tensor to cast.
   :type input: tl.tensor
   :param dtype: The target data type.
   :type dtype: tl.dtype
   :param fp_downcast_rounding: The rounding mode for downcasting floating-point values. This parameter is only used when ``input`` is a floating-point tensor and ``dtype`` is a floating-point type with a smaller bitwidth. Supported values are ``"rtne"`` (round to nearest, ties to even) and ``"rtz"`` (round towards zero).
   :type fp_downcast_rounding: str, optional
   :param bitcast: If true, the tensor is bitcasted to the given ``dtype``, instead of being numerically casted.
   :type bitcast: bool, optional
   :return: A tensor with the specified ``dtype``.
   :rtype: tl.tensor

   This function can also be called as a member function on :py:class:`tensor`, as ``x.cast(...)`` instead of ``cast(x, ...)``.

   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def kernel(x_ptr, y_ptr, BLOCK_SIZE: tl.constexpr):
          pid = tl.program_id(axis=0)
          offs = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)
          x = tl.load(x_ptr + offs)

          # Numerical cast: converts float32 values to int32 values
          y_int = tl.cast(x, tl.int32)

          # Bitcast: reinterprets float32 bits as int32 without changing values
          y_bits = tl.cast(x, tl.int32, bitcast=True)

          # Downcast with specific rounding mode
          y_fp16 = tl.cast(x, tl.float16, fp_downcast_rounding="rtne")

          tl.store(y_ptr + offs, y_int)
