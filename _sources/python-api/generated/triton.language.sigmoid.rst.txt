triton.language.sigmoid
======================

.. currentmodule:: triton.language

.. function:: sigmoid(x)

   Computes the element-wise sigmoid of :code:`x`.

   The sigmoid function is defined as :code:`1 / (1 + exp(-x))`.

   Parameters
   ----------
   x : Block
       The input tensor or scalar values.

   Returns
   -------
   Block
       A tensor containing the sigmoid of each element in :code:`x`.

   Notes
   -----
   This function can also be called as a member function on :py:class:`tensor`,
   as :code:`x.sigmoid()` instead of :code:`sigmoid(x)`.

   The sigmoid function maps input values to the range (0, 1) and is commonly
   used as an activation function in neural networks.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def kernel(x_ptr, output_ptr, n_elements, BLOCK_SIZE: tl.constexpr):
          pid = tl.program_id(axis=0)
          offsets = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)
          mask = offsets < n_elements
          x = tl.load(x_ptr + offsets, mask=mask)
          output = tl.sigmoid(x)
          tl.store(output_ptr + offsets, output, mask=mask)
