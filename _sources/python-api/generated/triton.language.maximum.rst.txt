Computes the element-wise maximum of two input tensors.

.. function:: triton.language.maximum(x, y, propagate_nan=tl.PropagateNan.NONE)

   Computes the element-wise maximum of :code:`x` and :code:`y`.

   If :code:`propagate_nan` is set to :code:`tl.PropagateNan.PROPAGATE_NAN`,
   NaN values in either input will propagate to the output. Otherwise, NaNs
   are treated as missing values and ignored (returning the non-NaN operand).

   For :code:`bfloat16` inputs, values are promoted to :code:`float32` before
   comparison.

   Parameters
   ----------
   x : tl.tensor
       The first input tensor.
   y : tl.tensor
       The second input tensor.
   propagate_nan : tl.PropagateNan, optional
       Whether to propagate NaN values. Default is :code:`tl.PropagateNan.NONE`.

   Returns
   -------
   tl.tensor
       A tensor containing the element-wise maximum values.

   See Also
   --------
   tl.minimum : Computes the element-wise minimum.
   tl.clamp : Clamps values within a range.
   tl.PropagateNan : Enum for NaN propagation behavior.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def kernel(X_ptr, Y_ptr, Out_ptr, BLOCK_SIZE: tl.constexpr):
          x = tl.load(X_ptr + tl.arange(0, BLOCK_SIZE))
          y = tl.load(Y_ptr + tl.arange(0, BLOCK_SIZE))
          out = tl.maximum(x, y)
          tl.store(Out_ptr + tl.arange(0, BLOCK_SIZE), out)
