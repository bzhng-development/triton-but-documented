randint4x
=========

.. jit_function:: triton.language.randint4x(seed, offset, n_rounds=10)

   Given a :code:`seed` scalar and an :code:`offset` block, returns four
   blocks of random :code:`int32`.

   This is the maximally efficient entry point to Triton's Philox pseudo-random
   number generator.

   Parameters
   ----------
   seed : tl.tensor
       The seed for generating random numbers.
   offset : tl.tensor
       The offsets to generate random numbers for. Supports both 32-bit and
       64-bit integers.
   n_rounds : tl.constexpr, optional
       Number of rounds for the Philox PRNG. Default is 10.

   Returns
   -------
   r0 : tl.tensor
       First block of random :code:`int32`.
   r1 : tl.tensor
       Second block of random :code:`int32`.
   r2 : tl.tensor
       Third block of random :code:`int32`.
   r3 : tl.tensor
       Fourth block of random :code:`int32`.

   Notes
   -----
   For generating multiple streams of random numbers, using :code:`randint4x`
   is more efficient than calling :code:`randint` four times.

   Examples
   --------
   .. code-block:: python

       @triton.jit
       def kernel(X, seed, offset):
           r0, r1, r2, r3 = tl.randint4x(seed, offset)
           # use r0, r1, r2, r3
