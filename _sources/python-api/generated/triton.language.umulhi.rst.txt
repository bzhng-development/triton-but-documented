umulhi
======

.. currentmodule:: triton.language

.. function:: umulhi(x, y, _semantic=None)

   Computes the element-wise most significant N bits of the 2N-bit product of :code:`x` and :code:`y`.

   Parameters
   ----------
   x : Block
       The first input tensor. Must be of type :code:`int32`, :code:`int64`, :code:`uint32`, or :code:`uint64`.
   y : Block
       The second input tensor. Must be of type :code:`int32`, :code:`int64`, :code:`uint32`, or :code:`uint64`.

   Returns
   -------
   Block
       A tensor containing the most significant N bits of the 2N-bit product of :code:`x` and :code:`y`. The dtype matches the input dtype.

   Notes
   -----
   For N-bit inputs, the full product is 2N bits. This function returns the upper N bits of that product. This is useful for high-precision multiplication where the full 2N-bit result is needed but only the high bits are required.

   Supported dtypes: :code:`int32`, :code:`int64`, :code:`uint32`, :code:`uint64`.

   Examples
   --------
   .. code-block:: python

      import triton
      import triton.language as tl

      @triton.jit
      def kernel(x_ptr, y_ptr, out_ptr, BLOCK_SIZE: tl.constexpr):
          pid = tl.program_id(0)
          offs = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)
          x = tl.load(x_ptr + offs)
          y = tl.load(y_ptr + offs)
          # Compute high 32 bits of 64-bit product
          hi = tl.umulhi(x, y)
          tl.store(out_ptr + offs, hi)
