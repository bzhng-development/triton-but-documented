unsqueeze
=========

Insert a new axis of size 1 at the specified dimension.

Parameters
----------
x : Block
    The input tensor.
dim : constexpr
    The dimension at which to insert the new axis. Must be a compile-time constant.

Returns
-------
Block
    A view of the input tensor with an additional dimension of size 1 inserted at :code:`dim`.

Notes
-----
This operation is equivalent to inserting a dimension of size 1 into the shape tuple at position :code:`dim`. The total number of elements remains unchanged.

Examples
--------
.. code-block:: python

    import triton.language as tl

    @triton.jit
    def kernel(X_ptr, Y_ptr, BLOCK_SIZE: tl.constexpr):
        x = tl.arange(0, BLOCK_SIZE)  # shape: (BLOCK_SIZE,)
        y = tl.unsqueeze(x, 0)        # shape: (1, BLOCK_SIZE)
        z = tl.unsqueeze(x, 1)        # shape: (BLOCK_SIZE, 1)
