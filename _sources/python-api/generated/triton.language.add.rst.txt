Add
===

.. currentmodule:: triton.language

.. function:: add(x, y, sanitize_overflow=True)

   Element-wise addition of two tensors.

   Parameters
   ----------
   x : tensor
       The first input tensor.
   y : tensor
       The second input tensor.
   sanitize_overflow : constexpr, optional
       If True (default), enables overflow sanitization. Set to False for
       potentially faster but unchecked arithmetic.

   Returns
   -------
   tensor
       A tensor containing the element-wise sum of :code:`x` and :code:`y`.

   Notes
   -----
   This function can also be called as a member function on :py:class:`tensor`,
   as :code:`x.add(y)` instead of :code:`add(x, y)`.

   The :code:`+` operator is overloaded to call this function, so
   :code:`x + y` is equivalent to :code:`tl.add(x, y)`.

   Examples
   --------
   .. code-block:: python

       import triton
       import triton.language as tl

       @triton.jit
       def add_kernel(x_ptr, y_ptr, output_ptr, n):
           pid = tl.program_id(0)
           offset = pid * tl.arange(0, 64)
           mask = offset < n
           x = tl.load(x_ptr + offset, mask=mask)
           y = tl.load(y_ptr + offset, mask=mask)
           z = tl.add(x, y)
           tl.store(output_ptr + offset, z, mask=mask)

       # Equivalent using the + operator:
       z = x + y

       # Equivalent using member function:
       z = x.add(y)
