triton.language.PropagateNan
============================

Enum controlling NaN propagation behavior in floating-point operations.

This enumeration specifies whether NaN inputs should propagate to outputs
for specific Triton language operations.

Attributes
----------

NONE
    Do not propagate NaNs; return a defined value (typically zero) if any
    input is NaN.

ALL
    Propagate NaNs; if any input is NaN, the output will be NaN.

Notes
-----
This enum is typically passed to the ``propagate_nan`` argument in
floating-point reduction or arithmetic operations to control IEEE 754
compliance behavior.

Examples
--------

.. code-block:: python

    import triton
    import triton.language as tl

    @triton.jit
    def kernel(x_ptr, y_ptr, out_ptr, n: tl.constexpr):
        pid = tl.program_id(0)
        x = tl.load(x_ptr + pid)
        y = tl.load(y_ptr + pid)
        
        # Force NaN propagation if any input is NaN
        out = tl.sqrt(x, propagate_nan=tl.PropagateNan.ALL)
        tl.store(out_ptr + pid, out)
