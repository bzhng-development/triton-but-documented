Returns the indices of the minimum values of a tensor along a given axis.

.. code-block:: python

    argmin(input, axis, tie_break_left=True, keep_dims=False)

Parameters
----------
input : tl.tensor
    Input tensor.
axis : int
    Axis along which to compute the argmin. If ``None``, reduction is performed over all dimensions.
tie_break_left : bool, optional
    If ``True`` (default), in case of ties, the left-most index is returned. If ``False``, the tie-breaking behavior is implementation-defined.
keep_dims : bool, optional
    If ``True``, the reduced axes are retained with length 1. Default is ``False``.

Returns
-------
tl.tensor
    Tensor of indices indicating the position of the minimum value.

Notes
-----
This function can also be called as a member function on :py:class:`tl.tensor`,
as ``x.argmin(...)`` instead of ``tl.argmin(x, ...)``.

The reduction operation requires associative and commutative semantics.

Examples
--------
.. code-block:: python

    import triton.language as tl

    # Example 1: Argmin over a specific axis
    @triton.jit
    def kernel(X, Y, stride):
        pid = tl.program_id(0)
        row = tl.arange(0, stride) + pid * stride
        x = tl.load(X + row)
        # Find index of minimum value in the row
        min_idx = tl.argmin(x, axis=0)
        tl.store(Y + pid, min_idx)

    # Example 2: Using tie_break_left=False
    @triton.jit
    def kernel_tie_break(X, Y):
        x = tl.load(X)
        # In case of ties, do not guarantee left-most index
        min_idx = tl.argmin(x, axis=0, tie_break_left=False)
        tl.store(Y, min_idx)
