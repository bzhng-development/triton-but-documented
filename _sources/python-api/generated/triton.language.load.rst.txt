.. py:function:: triton.language.load(pointer, mask=None, other=None, boundary_check=(), padding_option='', cache_modifier='', eviction_policy='', volatile=False)

   Load data from memory into a tensor.

   The behavior depends on the type of `pointer`:

   1. **Single element pointer**: Loads a scalar. `mask` and `other` must be scalars; `boundary_check` and `padding_option` must be empty.
   2. **Tensor of pointers**: Loads an N-dimensional tensor. `mask` and `other` are broadcast to `pointer.shape`; `boundary_check` and `padding_option` must be empty.
   3. **Block pointer** (from :py:func:`make_block_ptr`): Loads a tensor. `mask` and `other` must be `None`; `boundary_check` and `padding_option` control out-of-bounds access.

   Parameters
   ----------
   pointer : triton.PointerType or tensor of triton.PointerType
       Pointer(s) to the data to be loaded.
   mask : tensor of triton.int1, optional
       If ``mask[idx]`` is false, do not load data at ``pointer[idx]``. Must be ``None`` when using block pointers.
   other : tensor or scalar, optional
       Value to return if ``mask[idx]`` is false. Implicitly typecast to ``pointer.dtype.element_ty``.
   boundary_check : tuple of ints, optional
       Dimensions on which to perform boundary checks. Only valid with block pointers.
   padding_option : str, optional
       Padding value for out-of-bounds access. Options: ``""`` (undefined), ``"zero"``, ``"nan"``.
   cache_modifier : str, optional
       PTX cache operator. Options: ``""``, ``".ca"`` (cache at all levels), ``".cg"`` (cache at global/L2), ``".cv"`` (fetch without caching). See `NVIDIA PTX Cache Operators <https://docs.nvidia.com/cuda/parallel-thread-execution/index.html#cache-operators>`_.
   eviction_policy : str, optional
       PTX eviction policy.
   volatile : bool, optional
       If True, marks the load as volatile in PTX.

   Returns
   -------
   tensor
       The loaded data.

   Examples
   --------
   .. code-block:: python

       import triton
       import triton.language as tl

       @triton.jit
       def kernel(ptr, mask_ptr, BLOCK: tl.constexpr):
           # Load a block of data
           offsets = tl.arange(0, BLOCK)
           data = tl.load(ptr + offsets)

           # Load with masking
           mask = tl.load(mask_ptr + offsets)
           masked_data = tl.load(ptr + offsets, mask=mask, other=0.0)

           # Load using a block pointer (requires make_block_ptr)
           # block_ptr = tl.make_block_ptr(base=ptr, shape=(M, N), strides=(N, 1),
           #                               offsets=(0, 0), block_shape=(BLOCK, BLOCK), order=(0, 1))
           # blocked_data = tl.load(block_ptr, boundary_check=(0, 1), padding_option="zero")
