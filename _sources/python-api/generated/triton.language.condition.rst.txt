.. class:: triton.language.condition(arg1, disable_licm=False)

   While loop condition wrapper.

   This class is a special wrapper used to annotate ``while`` loops in the context of
   :code:`@triton.jit` functions. It allows users to pass extra attributes to the compiler
   to control optimization behavior.

   .. code-block:: python

      @triton.jit
      def kernel(...):
          while tl.condition(c, disable_licm=True):
              ...

   Parameters
   ----------
   arg1 : tensor
      The boolean condition tensor controlling the while loop execution.
   disable_licm : bool, optional
      If True, tells the compiler not to hoist loop-invariant code outside the loop.
      This is often useful to avoid creating long live ranges within a loop.
      Default is False.

   Notes
   -----
   This wrapper is specifically designed for use in the condition expression of
   ``while`` loops inside JIT-compiled Triton kernels. It does not perform any
   runtime computation but serves as a hint to the compiler backend.
