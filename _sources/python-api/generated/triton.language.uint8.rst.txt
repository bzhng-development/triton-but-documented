triton.language.uint8
=====================

8-bit unsigned integer data type.

This dtype represents an unsigned integer stored in 8 bits (1 byte), with a
value range of [0, 255]. It is used in Triton kernels to define the type of
pointers, tensors, or scalar values requiring compact storage and arithmetic
modulo 256.

Parameters
----------

None
    This is a dtype instance and does not accept arguments.

Returns
-------

dtype
    The ``uint8`` data type object, usable in ``tl.full``, ``tl.zeros``,
    pointer casting, and tensor type annotations.

Notes
-----

*   Arithmetic operations on ``uint8`` values wrap around modulo 256.
*   When mixed with larger integer types (e.g., ``int32``), standard promotion
    rules apply, typically promoting to the wider type.
*   Useful for memory-efficient storage of indices, masks, or quantized data.

Examples
--------

.. code-block:: python

    import triton
    import triton.language as tl

    @triton.jit
    def kernel(ptr, N):
        # Create a uint8 constant
        val: tl.uint8 = tl.constexpr(255)

        # Allocate a uint8 tensor
        buffer = tl.zeros([N], dtype=tl.uint8)

        # Perform wrapping arithmetic
        x: tl.uint8 = 200
        y: tl.uint8 = 100
        z = x + y  # Result is 44 (300 % 256)

        # Store result
        tl.store(ptr + tl.arange(0, N), z)
